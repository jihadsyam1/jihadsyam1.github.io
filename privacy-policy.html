Privacy Policy

آخر تحديث: مايو 2025

إضافة Smart Click & Type لا تجمع أو تخزن أي بيانات شخصية للمستخدمين. جميع السكربتات والموارد مضمنة داخل الإضافة ولا يتم تحميل أي كود خارجي عند التشغيل.

Last updated: May 2025

Smart Click & Type does not collect or store any personal user data. All scripts and resources are bundled within the extension; no external code is loaded at runtime.

الأذونات

  • activeTab: تمكن الإضافة من التفاعل مع التبويب الحالي عند النقر على أيقونتها.
  • scripting: تسمح بحقن الكود داخل صفحة الويب لتنفيذ أوامر النقر والكتابة تلقائيًا.

Permissions

  • activeTab: Allows the extension to interact with the current tab when you click the extension icon.
  • scripting: Enables injecting code into the webpage to perform automated click and type actions.

جمع البيانات

لا يتم جمع أي بيانات أو إرسالها إلى أي خادم. تعمل الإضافة بالكامل على متصفح المستخدم.

Data Collection

No data is collected or transmitted to any server. The extension runs entirely in the user’s browser.
